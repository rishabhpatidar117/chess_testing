# Chess Game in Browser

This is a fully functional chess game built using HTML, CSS, JavaScript, and `chess.js`.

### ✅ Features
- Human vs Human
- Human vs AI
- AI vs AI
- Valid move checking, check/checkmate detection
- PGN export/import support

### 🚀 Hosted on GitHub Pages
Visit: [https://yourusername.github.io/chess-game/](https://yourusername.github.io/chess-game/)

### 🧠 Tech Stack
- `chess.js`: Move validation and game rules
- `chessboard.js`: UI for the board
